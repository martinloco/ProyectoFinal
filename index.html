Actualizar paquetes


1. La siguiente instruccion ( sudo update ) sirve para identificar lo que ya estan instalados y lo que se debe actualizar en el ubuntu.

2. La siguiente instruccion ( sudo upgrate ) como lo dice el codigo sirve para actualizar cualquier paquete. Te pedira oprima la tecla ( s ) para confirmar la actualización.


MYSQL


1. Obligatoriamente tiene que estar conectado el dispositivo a internet. ( apt search mysql ) aparecera un listado de todas las apliacaciones que tiene instaladas.

2. Segundo paso (sudo apt install mysql-server ) y oprime la tecla enter para que comieze la descarga.

3. De la instruccion ( s ) para afirmar la descarga.


Apache


1. Tecleara el siguiente codigo ( apt search apache ) aparecera un listado de todas las apliacaciones que tiene instaladas.

2. Ya terminado el primer paso dara la instruccion (sudo apt install apache2) para que comieze la descarga.

3. Dara la instruccion ( s ) para afirmar la descarga.

4. En este paso aprendera a identificar si esta funcionando apache ( service --status-all | grep apache2 ) si aparece un [+]es que esta corriendo apache y si aparece [-]es que no esta corriendo.

6. Presionar las teclas ( ctrl+x ) para salir.

7. tecle el comando ( cat library.sql ) y dar enter para confirmar la salida del editor nano.

8. Escribir el comando ( sudo -i ) y dar enter para cambiar al usuario root.

9. Escribir ( mysql ) y dar enter para entrar a mysql.

10. Escribir el comando ( source /var/www/html/library.com/db/library.sql) y dar enter para crear la base de datos.

12. Escribir el comando ( select user host from mysql.user ) y dar enter para ver los usuarios.

2.Escribe en archivo el siguiente codigo

( VirtualHost*:80>

Server Alias www.library.com
Server Admin bryangfloresr2@gmail.com
DocumentRoot /var/log/library.com-error.log
CustomLog /var/log/library.com-acces.log. common

/VirtualHost )

3.Ya terminado el paso anterior tecle el comando ( ls ) .

4.En la carpeta ( sites-available ) y escribe ( sudo a2ensite library.com ) esto sirve para habilitar el dominio. Y reinicia el apache ( sudo service apache2 restart ).

5.Entra a la carpeta ( /etc/hosts/ ) y agrege el siguiente codigo
